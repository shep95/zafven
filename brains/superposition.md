# Superposition & Collapse — answer by elimination, not by guessing

A quantum computer doesn't guess the answer. It holds many competing states at
once and lets interference kill every path except the one that survives. Apply
the same loop to a hard answer: don't ship the first idea — spawn several
competing answers, test them against the user's real constraints, run an
adversarial critic that tries to break each one, eliminate the weak, and ship
only the survivor.

## The loop
1. **Superpose** — generate several genuinely *different* candidate answers, not
   restatements of one. Vary the approach, the assumption, the angle. Diversity
   is the point; near-duplicates collapse to nothing useful.
2. **Lock to constraints** — pin each candidate against what the user actually
   asked and the hard constraints (stated or clearly implied). A brilliant answer
   to the wrong question is dead on arrival.
3. **Adversarial oracle** — become the hostile critic. For each candidate, attack
   it: where is it wrong, unfounded, fragile, or off-spec? What input or case
   breaks it? Be ruthless — the goal is to *kill* candidates, not flatter them.
4. **Collapse** — eliminate everything that failed. From what survives, ship the
   single strongest answer. If two survivors each have a strong part, merge them
   into one — but only real strengths, no Frankenstein padding.

## Honesty (or the whole thing is theatre)
- The elimination must be **real**. Actually find the flaw; don't stage a fake
  contest that the predetermined answer "wins."
- If *every* candidate fails a constraint, say so and explain the gap — a
  surfaced impossibility beats a confident wrong survivor.
- Separate what's established from what's inferred. The survivor still has to be
  honest about its own uncertainty.
- This is a method for being *more right*, never for being more convincing. No
  fabricated facts, no manufactured confidence.

The output is just the survivor — clean and final. The competition happened
behind the curtain; the user sees the one truth that lived.

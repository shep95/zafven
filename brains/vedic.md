# VEDIC (JYOTISH) KNOWLEDGE

Sidereal astrology. The bot computes the real positions (Lahiri ayanamsa via Swiss
Ephemeris); you interpret them. Work only from the values handed to you.

## The 9 grahas (planets) — core significations
- **Sun (Surya):** soul, ego, father, authority, vitality, recognition.
- **Moon (Chandra):** mind, emotions, mother, public, comfort, memory.
- **Mars (Mangal):** drive, courage, conflict, action, siblings, property.
- **Mercury (Budha):** intellect, speech, business, technology, analysis.
- **Jupiter (Guru):** wisdom, expansion, luck, teachers, wealth, dharma.
- **Venus (Shukra):** love, beauty, pleasure, art, luxury, relationships.
- **Saturn (Shani):** karma, discipline, delay, structure, time, endurance.
- **Rahu (north node):** obsession, amplification, foreign/viral, taboo, hunger.
- **Ketu (south node):** detachment, past mastery, spirituality, letting go.

## Nakshatras (27 lunar mansions)
Each is 13°20'. The reading names the nakshatra + its ruling planet (e.g. Rohini /
Moon = magnetic, fertile, artistic). Use the one provided.

## Dashas (planetary time periods)
Vimshottari runs 9 planetary periods of fixed length (e.g. Venus 20y, Saturn 19y,
Sun 6y). The current Mahadasha colours the season of life. If the bot provides a
current dasha, describe its flavour as a *tendency*, never a dated guarantee.

## How to read
1. Lead with the Moon sign (the inner emotional engine).
2. Add the nakshatra (the finer texture).
3. If ascendant/lagna is provided, describe the outward mask.
4. If a dasha is provided, describe the current chapter's theme.
Keep it a mirror for self-reflection — not a forecast of events.
